# Personal Portfolio Website

Frontend Development Internship Project - CodeAlpha

---

# 📌 Project Description

This is a responsive Personal Portfolio Website developed using HTML and CSS. It showcases my profile, skills, projects, and contact information with a clean, modern, and user-friendly interface. The website includes an animated hero section with an automatic image slider and is fully responsive across different devices.

---

# 🚀 Features

- Responsive Design
- Modern Navigation Bar
- Animated Hero Section
- Automatic Image Slider
- About Me Section
- Skills Showcase
- Projects Section
- Contact Information
- Smooth Navigation
- Mobile Friendly Layout
- Clean and Attractive UI

---

# 🛠 Technologies Used

- HTML5
- CSS3
- Google Fonts
- CSS Flexbox
- CSS Animations
- Media Queries
